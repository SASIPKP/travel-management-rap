# Z_UI_TRAVEL_O4

## Object Type
Service Binding

## Binding Type
OData V4 - UI

## Service Definition
Z_UI_TRAVEL

## Entity Set

- Travel

## Status

Published

## Purpose

The Service Binding publishes the RAP business object as an OData V4 service.

It generates:

- OData V4 endpoint
- Service metadata
- Entity set (Travel)
- Preview application
- SAP Fiori Elements application integration

## Local Service Endpoint

/sap/opu/odata4/sap/z_ui_travel_o4/srvd/sap/z_ui_travel/0001/

## Fiori App URL

/sap/bc/ui5_ui5/sap/ztravel_ui_app

## Preview

The service binding provides a preview of the SAP Fiori Elements List Report and Object Page generated from the RAP business object.
